# 以太坊区块链浏览器 (Ethereum Chain Explorer)

该技能专注于对以太坊网络进行大规模、跨区块的数据查询与模式识别。

## 使用场景 (When to Use)
- **网络监控**: “现在的 Gas 费怎么样？基础费用 (Base Fee) 的趋势如何？”
- **账户画像**: 评估一个地址的活跃程度、首笔交易时间及其持有的所有资产。
- **批量事件检索**: “找出过去 1000 个区块中所有的巨额（>100 ETH）转账。”
- **大户追踪**: 监控特定“鲸鱼”地址的链上动态。

## 边界与衔接 (Boundaries)
- **如果需要深入分析某一个特定的错误交易** -> 移交给 `ethereum-transaction-inspector`。
- **如果发现目标地址是一个智能合约并需要分析其逻辑** -> 移交给 `ethereum-contract-analyzer`。

## 核心工作流 (Workflow)
1. **状态查询**: 获取最新区块、网络算力、Gas 价格分布。
2. **账户建模**: 综合 ETH 余额、代币余额及 Nonce 分析账户类型。
3. **范围检索**: 使用 `cast logs` 或 Etherscan API 在特定区块范围内检索事件。
4. **历史追踪**: 通过归档节点查询过去某个时刻的账户状态。
5. **流量分析**: 统计特定区块内各类交易（NFT, DeFi, Transfer）的占比。

## 推荐工具
- `cast`: 执行各类实时和历史的状态查询。
- Etherscan API: 用于获取大量的历史数据。

## 输出格式
- **统计摘要**: 指定范围内的汇总数据。
- **资产列表**: 账户持有的代币名称、余额及估值。
- **活动分析**: 最近 N 笔交易的行为模式。
- **网络状态**: Gas 费用分布及建议。
